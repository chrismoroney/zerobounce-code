Email Verification with ZeroBounce

Instructions:

  1. Upload a CSV file with format { Email (required), IP Address (if null, enter 'null') }
  2. While you see the loading circle in the browser, click the submit button to send your CSV to ZeroBounce API
  3. Click Download File to view the results from ZeroBounce.
  4. To use again, simply reload the page and submit another csv.
